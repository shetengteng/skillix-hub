安装指南

两种方式安装技能，按需选择

个人级安装

所有项目可用

# 克隆仓库
git clone https://github.com/shetengteng/skillix-hub.git
# 复制到 Cursor skills 目录
cp -r skillix-hub/swagger-api-reader ~/.cursor/skills/

项目级安装

仅当前项目可用

# 在项目根目录
mkdir -p .cursor/skills
cp -r skillix-hub/swagger-api-reader .cursor/skills/
提示： 安装后记得运行 pip install -r scripts/requirements.txt 安装依赖
