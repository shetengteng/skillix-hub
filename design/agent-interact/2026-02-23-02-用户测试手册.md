# Agent Interact 用户测试手册

> 日期：2026-02-23
> 前提：确保已安装依赖（`npm install` in `skills/agent-interact/` 和 `skills/agent-interact/ui/`）

## 准备工作

```bash
# 启动服务（自动打开浏览器）
node skills/agent-interact/tool.js start
```

启动后浏览器会自动打开 http://127.0.0.1:7890 ，页面显示"等待 Agent 发起交互..."即为正常。

---

## 场景 1：确认选择（confirm）

**用途**：Agent 需要用户从多个选项中选择一个。

```bash
node skills/agent-interact/tool.js dialog '{"type":"confirm","title":"请选择部署环境","message":"即将执行部署，请选择目标环境：","options":[{"id":"dev","label":"开发环境","description":"dev.example.com"},{"id":"staging","label":"预发布环境","description":"staging.example.com"},{"id":"prod","label":"生产环境","description":"prod.example.com"}],"timeout":120}'
```

**预期**：
- 浏览器弹出选择框，显示 3 个选项卡片
- 点击某个选项后高亮，点击确认
- 终端返回 `{"result":{"dialogId":"d-xxx","action":"dev"},...}`（action 为选中的选项 id）

---

## 场景 2：等待操作（wait）

**用途**：Agent 等待用户完成外部操作（如指纹验证、手动确认等）。

```bash
node skills/agent-interact/tool.js dialog '{"type":"wait","title":"等待指纹验证","message":"请在手机上完成指纹验证后点击确认","confirmText":"已完成验证","cancelText":"取消","timeout":120}'
```

**预期**：
- 浏览器弹出等待框，有脉冲动画
- 显示倒计时进度条
- 点击"已完成验证"→ 终端返回 `action: "confirmed"`
- 点击"取消"→ 终端返回 `action: "__cancelled"`

## 场景 3：图表展示（chart）

**用途**：Agent 需要可视化展示数据。

### 3a. 折线图

```bash
node skills/agent-interact/tool.js dialog '{"type":"chart","title":"API 响应时间趋势","chartType":"line","data":{"labels":["Mon","Tue","Wed","Thu","Fri","Sat","Sun"],"datasets":[{"label":"P50 (ms)","data":[45,52,48,61,55,42,38]},{"label":"P99 (ms)","data":[120,180,150,220,190,110,95]}]},"actions":[{"id":"close","label":"关闭"}],"timeout":120}'
```

### 3b. 饼图

```bash
node skills/agent-interact/tool.js dialog '{"type":"chart","title":"磁盘使用分布","chartType":"pie","data":{"labels":["代码","依赖","日志","缓存","其他"],"datasets":[{"label":"GB","data":[12,45,8,15,5]}]},"actions":[{"id":"ok","label":"了解"}],"timeout":120}'
```

### 3c. 柱状图

```bash
node skills/agent-interact/tool.js dialog '{"type":"chart","title":"各模块测试覆盖率","chartType":"bar","data":{"labels":["auth","api","db","cache","utils"],"datasets":[{"label":"覆盖率 %","data":[85,72,91,68,95]}]},"actions":[{"id":"ok","label":"了解"}],"timeout":120}'
```

**预期**：
- 浏览器弹出图表弹框，自动配色
- 点击操作按钮后终端返回对应 action

## 场景 4：通知提醒（notification）— 非阻塞

**用途**：Agent 推送通知，不需要用户响应。

### 4a. 成功通知

```bash
node skills/agent-interact/tool.js dialog '{"type":"notification","level":"success","title":"部署成功","message":"v2.0 已成功部署到生产环境","autoClose":8}'
```

### 4b. 警告通知

```bash
node skills/agent-interact/tool.js dialog '{"type":"notification","level":"warning","title":"磁盘空间不足","message":"剩余空间不足 10%，请及时清理","autoClose":8}'
```

### 4c. 错误通知

```bash
node skills/agent-interact/tool.js dialog '{"type":"notification","level":"error","title":"构建失败","message":"TypeScript 编译错误：3 个文件","autoClose":10}'
```

### 4d. 信息通知

```bash
node skills/agent-interact/tool.js dialog '{"type":"notification","level":"info","title":"任务开始","message":"正在分析代码库结构...","autoClose":5}'
```

**预期**：
- 终端**立即返回** `action: "delivered"`（不等待用户操作）
- 浏览器右上角滑入通知卡片
- 不同级别有不同图标和颜色（绿/黄/红/蓝）
- autoClose 秒后自动消失，或点击 × 关闭

## 场景 5：表单输入（form）

**用途**：Agent 需要用户填写结构化数据。

```bash
node skills/agent-interact/tool.js dialog '{"type":"form","title":"数据库连接配置","message":"请填写数据库连接信息：","fields":[{"id":"host","label":"主机地址","type":"text","default":"localhost","required":true},{"id":"port","label":"端口","type":"number","default":5432,"required":true},{"id":"database","label":"数据库名","type":"text","placeholder":"输入数据库名","required":true},{"id":"engine","label":"数据库类型","type":"select","options":["PostgreSQL","MySQL","SQLite"]},{"id":"notes","label":"备注","type":"textarea","placeholder":"可选备注信息"},{"id":"ssl","label":"SSL 连接","type":"checkbox","default":true}],"submitText":"连接","cancelText":"取消","timeout":120}'
```

**预期**：
- 浏览器弹出表单弹框
- 包含 6 个字段：文本输入、数字输入、下拉选择、文本域、复选框
- 必填字段标有红色 *
- 必填字段未填时"连接"按钮禁用
- 填写完成点击"连接"→ 终端返回 `action: "submitted"` + 表单数据
- 点击"取消"→ 终端返回 `action: "__cancelled"`

## 场景 6：审批门控（approval）

**用途**：Agent 执行敏感操作前需要用户审批。

### 6a. 低风险

```bash
node skills/agent-interact/tool.js dialog '{"type":"approval","title":"确认更新配置","message":"即将更新 staging 环境的配置文件","severity":"low","details":{"操作":"UPDATE CONFIG","环境":"staging","影响范围":"配置文件"},"timeout":120}'
```

### 6b. 高危操作（critical）

```bash
node skills/agent-interact/tool.js dialog '{"type":"approval","title":"确认删除数据库","message":"此操作将永久删除指定数据库，且不可恢复。","severity":"critical","details":{"操作":"DROP DATABASE","目标":"production_db","影响行数":"约 1,200,000 行","备份状态":"最近备份：2 小时前"},"approveText":"确认删除","rejectText":"取消","timeout":120}'
```

**预期**：
- 弹框显示严重级别标签（LOW/CRITICAL）
- details 以 key-value 表格展示
- critical 级别显示红色警告横幅"此操作不可逆，请谨慎确认"
- critical 级别的"确认删除"按钮为红色，**文字清晰可见**
- 点击"确认删除"→ 终端返回 `action: "approved"`
- 点击"取消"→ 终端返回 `action: "rejected"`

## 场景 7：进度展示（progress）

**用途**：展示多步骤任务的实时进度。

```bash
node skills/agent-interact/tool.js dialog '{"type":"progress","title":"项目部署进度","message":"正在执行自动化部署流程...","steps":[{"id":"deps","label":"安装依赖","status":"completed"},{"id":"build","label":"构建项目","status":"completed"},{"id":"test","label":"运行测试","status":"running"},{"id":"deploy","label":"部署到服务器","status":"pending"},{"id":"verify","label":"健康检查","status":"pending"}],"percent":45,"actions":[{"id":"ok","label":"了解"}],"timeout":120}'
```

**预期**：
- 弹框显示总进度条（45%）
- 5 个步骤列表，各有状态图标：
  - completed: 绿色勾
  - running: 蓝色旋转
  - pending: 灰色时钟
- 点击"了解"→ 终端返回 `action: "ok"`

## 场景 9：端口冲突

```bash
# 先占用 7890 端口
node -e "require('net').createServer().listen(7890, '127.0.0.1', () => console.log('占用 7890'))" &

# 启动服务（应自动切换到 7891）
node skills/agent-interact/tool.js start

# 清理
kill %1
```

**预期**：返回 `"Port 7890 occupied, started on http://127.0.0.1:7891"`

## 场景 10：服务管理

```bash
# 查看状态
node skills/agent-interact/tool.js status

# 停止服务
node skills/agent-interact/tool.js stop

# 再次查看状态
node skills/agent-interact/tool.js status
```

**预期**：
- 运行中返回 `{"status":"running","port":7890,"activeDialogs":0}`
- 停止后返回 `{"status":"stopped"}`
