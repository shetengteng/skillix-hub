# Skill Store 自然语言使用指南

## 工作原理

Skill Store 通过 SKILL.md 中的"自主决策指南"和"触发词"引导 AI Agent 自动匹配用户意图，调用对应的 CLI 命令链。用户无需记忆命令，直接用自然语言描述需求即可。

## 前置条件

- Skill Store 已全局安装到 `~/.cursor/skills/skill-store`
- 会话 Hook（`skill-store-hook.mdc`）已注册

## 自然语言 Demo

### Demo 1：添加仓库源

**用户说：**
> 添加 skillix-hub 仓库 https://github.com/shetengteng/skillix-hub.git

**AI 自动执行：**
```
registry.py add --url "https://github.com/shetengteng/skillix-hub.git" --alias "skillix-hub"
  → sync.py all
  → index.py rebuild
```

### Demo 2：搜索 Skill

**用户说：**
> 我需要一个浏览器自动化的 Skill

**AI 自动执行：**
```
index.py search --query "浏览器自动化"
  → 展示搜索结果，推荐最匹配的 Skill
```

### Demo 3：安装 Skill

**用户说：**
> 安装 playwright

**AI 自动执行：**
```
install.py install --name "playwright" --scope global
  → 自动解析依赖
  → 安装完成
```

### Demo 4：查看状态

**用户说：**
> 查看已安装的 Skill

**AI 自动执行：**
```
install.py list
  → 展示已安装的 Skill 列表
```

### Demo 5：更新

**用户说：**
> 更新所有 Skill

**AI 自动执行：**
```
install.py update-all
  → 逐个检查并更新
```

### Demo 6：卸载

**用户说：**
> 卸载 playwright

**AI 自动执行：**
```
install.py uninstall --name "playwright"
```

## 简单测试流程

在 Cursor 中开启新会话，依次输入以下自然语言指令：

```
1. "添加仓库 https://github.com/shetengteng/skillix-hub.git"
   → 验证：AI 调用 registry add + sync + index rebuild，输出 JSON 结果

2. "搜索 memory"
   → 验证：AI 调用 index search，返回匹配的 Skill 列表

3. "列出所有可用的 Skill"
   → 验证：AI 调用 index list，展示全部 Skill

4. "查看已安装的 Skill"
   → 验证：AI 调用 install list

5. "移除 skillix-hub 仓库"
   → 验证：AI 调用 registry remove，清理测试数据
```

## CLI 快速验证（不依赖 AI Agent）

```bash
cd ~/.cursor/skills/skill-store

# 添加仓库
python3 scripts/registry.py add --url "https://github.com/shetengteng/skillix-hub.git" --alias "skillix-hub"

# 构建索引
python3 scripts/index.py rebuild

# 搜索
python3 scripts/index.py search --query "memory"

# 列出所有
python3 scripts/index.py list

# 查看已安装
python3 scripts/install.py list

# 清理
python3 scripts/registry.py remove --alias "skillix-hub"
```

所有命令输出 JSON 格式，可直接验证结果正确性。
